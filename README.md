# Todo List Terminal em Python

Este projeto é uma aplicação de lista de tarefas desenvolvida em Python para ser executada no terminal.

A proposta do projeto é mostrar uma evolução gradual no desenvolvimento de uma aplicação simples, começando com uma versão que armazena os dados apenas em memória e evoluindo para uma versão com persistência em arquivo JSON.

## Objetivo do projeto

O objetivo principal é praticar conceitos fundamentais de programação, organização de código e evolução de projeto para portfólio no GitHub.

## Versões do projeto

### Versão 1 - Simples

Nesta primeira versão, as tarefas são armazenadas apenas em memória.

Funcionalidades:

- Adicionar tarefas
- Listar tarefas
- Marcar tarefas como concluídas
- Editar tarefas
- Remover tarefas
- Trabalhar com classes, funções e listas

Pasta:

versao_1_simples/
Versão 2 - Persistência com JSON

Nesta segunda versão, foi adicionada a persistência dos dados em um arquivo JSON.

Funcionalidades adicionadas:

Salvar tarefas em arquivo JSON
Carregar tarefas ao iniciar o programa
Separar tarefas pendentes e concluídas na listagem
Manter os dados mesmo após fechar o programa

Pasta:

versao_2_json/
Tecnologias utilizadas
Python
JSON
Programação orientada a objetos
Manipulação de arquivos
Terminal
Como executar

Escolha uma das versões e acesse a pasta correspondente.

Exemplo:

cd versao_2_json
python main.py
